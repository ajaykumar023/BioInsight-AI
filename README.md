# 🧬 BioInsight-AI

### Explainable Artificial Intelligence for Heart Disease Prediction

An AI-powered clinical decision support system that predicts the likelihood of heart disease using Machine Learning and explains every prediction with SHAP Explainable AI.

## 📌 Overview

BioInsight-AI is an Explainable AI application developed to assist in heart disease risk prediction using machine learning.

Unlike traditional prediction systems, BioInsight-AI not only predicts whether a patient is at risk but also explains **why** the prediction was made using SHAP (SHapley Additive Explanations).

The application provides interactive visualizations, feature importance analysis, and downloadable PDF reports, making the prediction process transparent and easy to understand.
